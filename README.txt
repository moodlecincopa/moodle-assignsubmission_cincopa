Cincopa plugin for Assign submission
====================================


Requirements
============

- Moodle 3.1


Installation
============
First way:

 1. Place the "cincopa" folder within your_moodle_install/mod/assign/submission/
 2. Login as an administrator, visit the notifications page and complete the installation

Second way:

 1. Login as an administrator navigate to Site administration -> Plugins -> Install plugins
 2. Choose ZIP file from your device 
 3. The ZIP will be extracted into an appropriate location for the plugin type
 4. After that, you can complete the installation
